# Kanto Shiny Checklist

A self-contained deluxe tracker for all 151 Kanto shiny Pokémon.

## Features

- Saves checklist progress in browser `localStorage` under `kanto_shiny_checklist_v1`.
- Keeps view settings in a separate `kanto_shiny_checklist_settings_v1` key.
- Preserves the original import/export save shape.
- Runs as a static GitHub Pages site with no build step.
- Loads shiny sprites from the PokéAPI sprites GitHub repository.

## Local Preview

Open `index.html` directly, or serve the folder:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.
